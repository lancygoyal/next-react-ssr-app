
## React App

React.js App Powered By Next.js and Redux.

### Includes

- React.js
- Redux
- Next.js
- Material-UI
- Redux-Persist
- Typescript
- Jest

### Running it

```
npm i
npm run dev
```

### Testing it

```
npm run test
```

### Folder structure

    .
    ├── __test__				# unit test cases
    │   ├── components
    ├── components              # reuseable components
    ├── config					# app configs
    ├── constants
	├── pages               	# app routes
    │   ├── api					# api
    │   ├── index.js
	├── redux               	# redux setup & modules
    └── package.json            # node dependencies
